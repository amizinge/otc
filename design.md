# OTC Crypto Platform Design System

## Design Philosophy

**Visual Language**: Modern financial platform aesthetic inspired by Binance and professional trading interfaces. Clean, data-driven design with emphasis on trust, security, and efficiency. Sophisticated color palette that conveys professionalism while remaining accessible to Nigerian market.

**Color Palette**: 
- Primary: Deep Navy (#1a1d29) for backgrounds and headers
- Secondary: Warm Gold (#f7931a) for CTAs and highlights (Bitcoin-inspired)
- Accent: Electric Blue (#00d4ff) for links and active states
- Success: Emerald Green (#10b981) for positive values
- Warning: Amber (#f59e0b) for alerts and pending states
- Error: Coral Red (#ef4444) for errors and negative values
- Text: Pure White (#ffffff) and Light Gray (#e5e7eb) for readability
- Card Backgrounds: Dark Slate (#2d3748) with subtle transparency

**Typography**: 
- Display Font: "Inter" - Modern, clean sans-serif for headings and UI elements
- Body Font: "JetBrains Mono" - Monospace for numerical data and prices
- Font Hierarchy: Large bold headings, medium-weight body text, small monospace for data

## Visual Effects & Styling

**Used Libraries**:
- Anime.js: Smooth transitions and micro-interactions
- ECharts.js: Real-time trading charts and market data visualization
- Pixi.js: Particle effects for background animations
- Typed.js: Dynamic text effects for hero sections
- Splitting.js: Text animation effects
- Splide.js: Image carousels and content sliders
- Matter.js: Physics-based animations for decorative elements

**Background Effects**: 
- Liquid-metal displacement shader using WebGL
- Subtle particle system with floating geometric shapes
- Gradient mesh overlays with low opacity
- Depth parallax layers for visual hierarchy

**Animation & Interactions**:
- Smooth hover effects with 3D transforms on cards
- Real-time price ticker animations
- Loading states with skeleton screens
- Micro-interactions for buttons and form elements
- Scroll-triggered animations with stagger effects

**Header Effects**:
- Dynamic navigation with blur backdrop
- Real-time market data ticker
- Animated logo with subtle glow effects
- Responsive hamburger menu with smooth transitions

**Data Visualization**:
- Candlestick charts for trading data
- Real-time price charts with smooth animations
- Market depth visualization
- Portfolio performance graphs
- Transaction history timelines

**Card Design**:
- Glassmorphism effects with backdrop blur
- Subtle border glow on hover
- Layered shadows for depth
- Consistent padding and border radius
- Gradient overlays for premium features

**Interactive Elements**:
- Button hover effects with scale and glow
- Form field focus states with accent colors
- Toggle switches with smooth transitions
- Dropdown menus with fade animations
- Modal overlays with backdrop blur

**Mobile Responsiveness**:
- Adaptive layouts for different screen sizes
- Touch-friendly interface elements
- Optimized typography scaling
- Simplified navigation for mobile
- Gesture-based interactions
